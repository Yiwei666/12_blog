## 3. Sublime text

### 1. md文件大纲视图预览

你可以安装“Outline”插件来实现右侧实时大纲视图。具体步骤如下：

1. 打开 Sublime Text，按 `Ctrl+Shift+P` 调出命令面板，输入 `Package Control: Install Package`，回车。

2. 在搜索框中输入 `Outline` 并回车安装即可。

3. 安装完成后，再次按下 `Ctrl+Shift+P`，输入 `Browse Mode: Outline (Right)` 并回车，就会在编辑窗口右侧出现大纲面板，自动列出当前 Markdown 文件中的所有 # 标题，编辑时会实时更新。

4. 关闭大纲视图时，点击面板右上角的关闭按钮，或按 `Shift+Alt+1` 恢复单窗格布局即可。

5. 再次打开大纲视图，只需按 `Ctrl+Shift+P` 调出命令面板，输入并选择 “Outline: Toggle Outline (Right)”（或直接输入“Outline”找到相应命令）即可在右侧重现大纲。

## 1. 浏览器

| 排名 | 浏览器 | 说明 | 官网下载网址 |
|---:|---|---|---|
| 1 | **Chrome** | 主力浏览器，兼容性最强，扩展生态丰富，网页兼容性几乎完美，适合办公、查资料、文献、日常使用。 | [Google Chrome 官网](https://www.google.com/chrome/) |
| 2 | **Firefox** | 独立内核 Gecko，隐私保护好，安全性高，适合测试网页、隐私需求或扩展 Chrome 无法满足的场景。 | [Mozilla Firefox 官网](https://www.mozilla.org/firefox/new/) |
| 3 | **Brave** | 隐私优先，默认拦截广告与追踪器；同样基于 Chromium 内核，兼容性很好，适合不想被追踪或看网页不想广告打扰时使用。 | [Brave 官网](https://brave.com/download/) |
| 4 | **Edge** | 系统自带 Chromium 内核浏览器，兼容性不错，Windows 集成度高，适合打开微软服务或作为备用浏览器；隐私控制一般。 | [Microsoft Edge 官网](https://www.microsoft.com/edge/download) |
| 5 | **Vivaldi** | 功能极度可定制，适合重度用户管理大量标签和工作流；隐私保护不错，但占用内存略高，不适合作为唯一浏览器。 | [Vivaldi 官网](https://vivaldi.com/download/) |
| 6 | **Opera** | 内置浏览器级免费 VPN（仅代理浏览器流量，非系统级）及广告拦截；自有扩展商店规模较小，但支持直接安装 Chrome 扩展；适合偶尔备用或轻度隐私使用。 | [Opera 官网](https://www.opera.com/download) |

# 电脑组装配置（二）


# 1. windows系统安装

### 1. 安装前准备

在安装 Windows 11 专业版之前，需要准备：

| 项目 | 说明 |
|---|---|
| U 盘 | 建议 16GB 或以上，制作安装盘时会被清空 |
| 旧电脑/笔记本 | 用于下载并制作 Windows 11 安装 U 盘 |
| 新电脑 | 已完成硬件组装，并确认可以点亮进入 BIOS |
| 键盘鼠标 | 建议插在主板后置 USB 接口 |
| 显示器 | 显示器 DP/HDMI 线应连接到独立显卡接口，而不是主板接口 |

新电脑第一次点亮时，如果能进入 BIOS，并能识别 CPU、内存、硬盘，说明核心硬件基本正常。


### 2. 制作 Windows 11 安装 U 盘

> 首先在旧电脑上打开微软 Windows 11 [官方下载页面](https://www.microsoft.com/en-us/software-download/windows11)，下载 `Media Creation Tool`，也就是 Windows 11 安装媒体创建工具。

> 注意不要下载或复制`Windows 11 安装助手`。安装助手主要用于已有 Windows 系统的在线升级，而新电脑全新安装系统应使用 `Media Creation Tool` 制作启动 U 盘。

操作流程如下：

```
1. 插入 U 盘
2. 运行 Media Creation Tool
3. 选择“为另一台电脑创建安装介质”
4. 语言选择：中文（简体）
5. 版本选择：Windows 11
6. 体系结构选择：64 位
7. 介质选择：U 盘
8. 等待工具自动下载并制作完成
```

制作工具界面中只显示 “Windows 11”，不会提前显示“Windows 11 专业版”。


### 3. 从 U 盘启动新电脑

1. 制作好安装 U 盘后，将 U 盘插入新电脑主板后置 USB 接口。

2. 开机后连续按：`F11`

3. 进入 MSI 主板的启动菜单，然后选择类似下面的启动项：`EFI: U盘名称`。建议选择带有 UEFI 字样的 U 盘启动项。


### 4. 开始安装 Windows 11

1. 进入 Windows 安装界面后，依次选择：

```
语言：中文（简体）
时间和货币格式：中国
键盘输入法：微软拼音
```

如果系统询问产品密钥，可以选择：`我没有产品密钥`

2. 之后在版本列表中选择：

```
Windows 11 Pro
或
Windows 11 专业版
```

注意这里不要选家庭版。可以先不激活，进入系统后再输入专业版密钥激活。


### 5. 选择安装方式和硬盘

1. 安装类型选择：`自定义：仅安装 Windows（高级）`

不要选择“升级”，因为这是新电脑全新安装系统。


2. 在硬盘选择界面中，如果是新硬盘，会看到类似：

```
驱动器 0 未分配空间
容量约 1.86TB
```

本次使用的是 2TB NVMe SSD，Windows 中显示约 1.81TB 或 1.86TB 属于正常现象。

最简单稳妥的做法是：

```
选择“驱动器 0 未分配空间”
点击“下一步”
```

Windows 会自动创建 EFI 系统分区、MSR 保留分区、C 盘主分区和恢复分区。

如果后续想分成 C 盘和 D 盘，不一定要在安装时手动分区，可以先让 Windows 自动安装，进入系统后再用“磁盘管理”压缩 C 盘并新建 D 盘。


### 6. 等待系统自动安装

接下来 Windows 会自动完成：

```
复制文件
安装功能
安装更新
自动重启
```

电脑重启后，不要反复按 F11，否则可能再次进入 U 盘安装界面。

如果重启后又进入安装程序，可以拔掉 U 盘，或者在 BIOS/启动菜单中选择硬盘启动。

# 3. 电脑压测和记录

## 1. 系统与硬件监控：`hwinfo`

> HWiNFO 是一款专业的系统信息与硬件监控工具，支持 Windows 和 DOS 平台，能够深度检测 CPU、GPU、主板、内存、存储等几乎所有硬件组件的详细参数。
> 
> 它提供实时传感器监控功能，可追踪温度、电压、风扇转速、功耗等数据，广泛用于硬件诊断、超频调试和系统稳定性测试。由于其数据准确、覆盖全面，HWiNFO 深受硬件爱好者、技术人员和游戏玩家的青睐。

官网下载：https://www.hwinfo.com/download/

注意：

- 强烈建议选择 Local (U.S.) 或者 FOSSHUB 节点下的 "Portable"（绿色便携版）。
- 便携版下载下来是一个压缩包，解压后直接双击就能运行，不需要安装到系统里，干净清爽，测试完不想用随时可以删掉。
- 双击 `HWiNFO64.exe` 后，会弹出一个小的启动窗口，勾选 "Sensors-only"（仅传感器），这个模式会直接打开包含温度、电压、频率和功耗的详细表格。
- 可以一边开着 HWiNFO64，一边运行压测软件（如 AIDA64 或 Cinebench），它会实时记录下最高温度和平均温度。

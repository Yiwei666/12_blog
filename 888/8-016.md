# Xray服务器端安装

## 1. 安装和更新

- 安装并升级 Xray-core 和地理数据，默认使用 User=nobody，但不会覆盖已有服务文件中的 User 设置

```sh
bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install

# 设置 Xray 开机自启
systemctl enable xray

# 检查 Xray 运行状态
systemctl status xray
```


- xray配置文件路径

```sh
/usr/local/etc/xray/config.json
```


- 常用命令

```sh
# 查看当前进程
ps -ef | grep -E 'v2ray|xray' | grep -v grep

# 查看端口占用
ss -ltnp | grep -E 'v2ray|xray'

# 查看已有配置路径
systemctl cat v2ray
systemctl cat xray

```

## 2. 配置文件

### 1. xray常用命令行

```sh
# 查看当前安装的 Xray 版本
xray version

# 生成 UUID
xray uuid

# 生成 REALITY 所需的 X25519 密钥对
xray x25519

# 从已有私钥推导公钥
xray x25519 -i "你的私钥内容"

# 校验配置文件语法，不启动服务
xray run -test -c /usr/local/etc/xray/config.json

# 手动以前台方式运行 Xray，用于实时调试
xray run -c /usr/local/etc/xray/config.json

# 输出合并后的配置，适合多配置文件排查
xray run -dump -c /usr/local/etc/xray/config.json

# 生成临时 TLS 证书
xray tls cert

# 测试目标域名的 TLS 握手
xray tls ping example.com

# 计算 TLS 证书链 hash
xray tls certChainHash
```
